# Design 100 Reference — AI 圖卡風格靈感庫

收錄 100 種設計風格，涵蓋日系經典與國際通用風格。每種風格都附有 AI 生成的圖卡範例與可直接複製的繁體中文風格描述，貼入 ChatGPT 或 Google Gemini 即可生成對應風格的知識型圖卡。

**Live Site**: https://vizance.github.io/100designReference/

## 這是什麼？

這是一個為「AI 圖卡生成課程」設計的風格參考工具。很多人想用 AI 製作知識型社群圖卡，但不知道怎麼描述自己想要的視覺風格。這個網站讓你：

1. 瀏覽 100 種風格的實際圖卡範例
2. 找到喜歡的風格後，一鍵複製風格描述
3. 將描述貼入 ChatGPT 或 Google Gemini，生成你自己的圖卡

## 風格分類

| 分類 | 數量 | 說明 |
|------|------|------|
| 圖卡排版 | 20 | 粗體排版、極簡留白、照片拼貼等排版導向風格 |
| 傳統日式 | 14 | 浮世繪、水墨畫、枯山水、金繼等經典和風 |
| 現代插畫 | 18 | 平面插畫、水彩、手帳、食物插畫等 |
| 動漫次文化 | 12 | 賽璐璐動畫、少年漫畫、像素藝術、Lo-Fi 等 |
| 設計風格 | 15 | 日式極簡、包浩斯、波普藝術、故障藝術等 |
| 科技未來 | 13 | 霓虹燈、3D 渲染、暗黑主題、電路板等 |
| 自然有機 | 8 | 櫻花、紅葉、竹林禪意、森林療癒等 |

## 功能

- **分類篩選** — 按 7 大風格分類瀏覽
- **場景篩選** — 按適用場景過濾（知識學習、商業職場、生活風格、身心靈、科技創業等）
- **關鍵字搜尋** — 支援中文、日文、英文搜尋
- **圖片放大** — 點擊圖卡預覽可放大檢視
- **一鍵複製** — 複製繁體中文風格描述，直接貼入 AI 使用
